Rules to Play:
  • When a key is pressed in server (instruction), you will receive it; and a timer counting up to 10 seconds in which the same instruction has to be sent.
  • The server verifies if the instruction sent by you matches “the instruction” and assign +1 point on correct, -1 on a wrong key and 0 for timeouts.
  • The game is over when the score reaches either +10 points or -3 points.
  • The game is over if you do not respond for 3 continuous instructions.
